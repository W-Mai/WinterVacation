变量命名规范
1. 标识符要能体现实际含义（这一条也可以为命名提供思路，毕竟命名是编程中最难的事情，hhh）
2. 变量名用小写字母
3. 用户自定义的类名一般以大写字母开头，
4. 对于多个单词组成的标识符，采用小驼峰命名法或下划线命名法
